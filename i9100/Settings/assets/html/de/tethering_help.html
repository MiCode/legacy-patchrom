USB-Tethering

Ihr Computer kann die Internetverbindung Ihres Geräts mitnutzen, wenn Sie die beiden Geräte über ein USB-Kabel verbinden.
  • Tethering funktioniert mit Windows 7, Windows Vista und Linux.

  • Sie können während des Tetherings keine SD-Karte mit Ihrem Computer verbinden.

  • Weitere Informationen, einschließlich Informationen zur Verwendung von USB-Tethering mit anderen Betriebssystemen, finden Sie unter http://www.android.com/tether#usb.

Mobiler WLAN-Hotspot

Sie können Ihr Gerät als mobilen WLAN-Hotspot nutzen und die mobile Internetverbindung so mit anderen Geräten teilen..
  • Es ist nicht möglich Geräteapplikationen für eine Internetverbindung via WLAN zu nutzen, solange das Gerät als mobiler WLAN-Hotspot genutzt wird.

  • Sie können die Einstellungen des mobilen WLAN-Hotspots in den WLAN-Hotspot-Einstellungen konfigurieren.

  • Besuchen Sie http://www.android.com/tether#wifi für weitere Informationen.
