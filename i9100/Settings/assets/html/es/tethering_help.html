Anclaje a red de USB

El equipo puede compartir la conexión a Internet de su dispositivo si los ancla mediante un cable USB
  • El anclaje funciona con Windows 7, Windows Vista y Linux

  • No se puede conectar una tarjeta SD al equipo cuando está anclado

  • Para obtener más información, incluida información acerca de cómo utilizar el anclaje a red de USB con otros sistemas operativos, diríjase a http://www.android.com/tether#usb.

Zona activa de WLAN portátil

Puede convertir su dispositivo en una zona activa de WLAN portátil para compartir la conexión a Internet con equipos u otros dispositivos
  • No se pueden utilizar aplicaciones del dispositivo para acceder a Internet mediante una conexión WLAN mientras el dispositivo actúa como zona activa de WLAN portátil

  • Puede configurar la zona activa de WLAN portátil en la configuración de zona activa de WLAN

  • Para más información, visite http://www.android.com/tether#wifi
