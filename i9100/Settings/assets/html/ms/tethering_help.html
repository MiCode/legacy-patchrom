Penambatan USB

Komputer anda boleh berkongsi sambungan Internet peranti anda jika anda menambatkannya bersama dengan mengguna kabel USB
  • Penambatan dapat berfungsi dengan Windows 7, Windows Vista, dan Linux

  • Anda tidak boleh menyambung kad SD ke komputer anda bila ditambat

  • Untuk maklumat selanjutnya, termasuk bagaimana mengguna penambatan USB dengan sistem operasi lain, pergi ke http://www.android.com/tether#usb

Titik panas WLAN mudah alih

Anda boleh menukar peranti anda ke titik panas WLAN, untuk berkongsi sambungan Internet dengan komputer atau peranti lain
  • Tidak boleh mengguna aplikasi peranti untuk mencapai Internet melalui sambungan WLAN bila peranti bekerja sebagai titik panas WLAN mudah alih

  • Anda boleh menatarajah titik panas WLAN mudah alih dalam aturan titik panas WLAN

  • Untuk maklumat selanjutnya, melawati http://www.android.com/tether#wifi
