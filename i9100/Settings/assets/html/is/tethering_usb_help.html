USB-netaðgangur

Tölvan þín getur notað nettengingu tækisins ef þú tengir símann við hana með USB-snúru
  • Sími sem netaðgangsstaður virkar með Windows 7, Windows Vista og Linux

  • Ekki er hægt að tengja SD-kort við tölvuna þegar sími er notaður sem netaðgangsstaður

  • Nánari upplýsingar um hvernig USB-nettengingar eru notaðar með öðrum stýrikerfum eru á http://www.android.com/tether#usb
