USB piesaiste

Jūsu dators var koplietot ierīces interneta savienojumu, ja vien starp datoru un ierīci izveidosiet savienojumu, izmantojot USB kabeli
  • Piesaistīšana darbojas operētājsistēmā Windows 7, Windows Vista un Linux

  • Piesaistes laikā datoram nevar pievienot SD karti

  • Lai iegūtu plašāku informāciju, kā arī lai uzzinātu, kā lietot USB piesaisti citās operētājsistēmās, apmeklējiet vietni http://www.android.com/tether#usb

Pārvietojams WLAN tīklājs

Varat pārvērst savu ierīci par pārvietojamu WLAN tīklāju, lai koplietotu tā interneta savienojumu ar datoriem vai citām ierīcēm
  • Ja ierīce kalpo kā pārvietojams WLAN tīklājs, nevar lietot ierīces programmas, lai piekļūtu internetam, izmantojot WLAN savienojumu

  • Varat konfigurēt pārvietojamo WLAN tīklāju WLAN tīklāja iestatījumos

  • Lai iegūtu plašāku informāciju, apmeklējiet http://www.android.com/tether#wifi
