Nascaireacht USB

Féadann do ríomhaire nasc Idirlín do ghléis a chomhroinnt má dhéanann tú nascaireacht eatarthu le cábla USB
  • Oibríonn nascaireacht le Windows 7, Windows Vista agus Linux

  • Ní féidir leat cárta SD a cheangal le do ríomhaire le linn nascaireachta

  • Le tuilleadh faisnéise a fháil, lena n-áirítear conas nascaireacht USB a úsáid le córais oibriúcháin eile, gabh chuig http://www.android.com/tether#usb

Ball nasctha WLAN inaistrithe

Féadann tú ball te inaistrithe WLAN a dhéanamh de do ghléas, chun a nasc Idirlín a chomhroinnt le ríomhairí nó le gléasanna eile
  • Ní féidir feidhmchláir an ghléis a úsáid chun an tIdirlíon a rochtain trí nasc WLAN agus an gléas ag obair mar bhall te inaistrithe WLAN

  • Féadann tú ball te inaistrithe WLAN a chumrú sna socruithe bhall te WLAN

  • Chun tuilleadh eolais a fháil, tabhair cuairt ar http://www.android.com/tether#wifi
