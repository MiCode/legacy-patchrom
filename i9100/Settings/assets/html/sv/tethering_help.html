USB-internetdelning

Du kan dela enhetens Internetanslutning med datorn om du kopplar samman dem med USB-kabeln
  • Internetdelning fungerar med Windows 7, Windows Vista och Linux

  • Du kan inte ansluta SD-kortet till datorn vid Internetdelning

  • Mer information inklusive information om hur du använder Internetdelning med USB med andra operativsystem finns på http://www.android.com/tether#usb

Portabel WLAN-hotspot

Du kan förvandla din enhet till en portabel WLAN-hotspot och dela dess Internetanslutning med datorer och andra enheter
  • Det går inte att använda enhetsprogram för att få åtkomst till Internet via WLAN-anslutning om enheten används som portabel WLAN hotspot

  • Du kan konfigurera portabel WLAN-hotspot i inställningarna för WLAN-hotspot

  • Mer information finns på http://www.android.com/tether#wifi
