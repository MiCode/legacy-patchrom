Tethering USB

Potete condividere la connessione dati del dispositivo con il PC collegandoli con un cavo USB
  • Tethering USB è supportato da Windows 7, Windows Vista e Linux

  • Non potete utilizzare la scheda SD durante il tethering

  • Per ulteriori informazioni, incluse quelle relative al collegamento tramite tethering USB con altri sistemi operativi, visitate il sito http://www.android.com/tether#usb.

Router WLAN

Potete condividere la connessione dati con altri dispositivi
  • Non potete collegare il dispositivo ad una rete WLAN quando la funzione Router WLAN è attiva

  • Potete configurare Router WLAN nel menu Impostazioni -> Wireless e rete -> Tethering e Router WLAN

  • Per ulteriori informazioni, visitate http://www.android.com/tether#wifi.
